# php-laravel-app-example1-login

## (local)本地環境隔離快速專案部屬(隨機PORT)
需安裝Docker(Windows、Mac、Linux上均可安裝), 若在Linux環境需額外手動安裝docker-compose, 部屬結果與UI相同
``` 
docker-compose up -d --build 
```
部屬包含laravel網頁+mariadb, 驗證後即可上傳程式碼
